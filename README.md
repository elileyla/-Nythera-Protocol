# Nythera Protocol

Nythera Protocol is a modular Layer 1 blockchain written in Rust, designed for deterministic settlement and AI-native decentralized systems.

## Network Parameters

- Consensus: Proof of Stake (PoS)
- Block Time: 1.8 seconds
- Finality: ~6 seconds
- Initial Supply: 21,500,000 NYT
- Max Supply: 215,000,000 NYT
- Initial Inflation: 7%
- Halving Cycle: 4 years

## Architecture

Nythera separates:
- Networking Layer
- Consensus Engine
- Execution Runtime
- Governance
- Staking
